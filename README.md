Real-time departure monitoring system for Dagupan City waterways, built with Laravel, React
(Vite/TypeScript), MySQL, and the Semaphore SMS API. Lets boat operators on basic keypad
phones report departures via SMS, with live updates surfaced to commuters through a PWA.
